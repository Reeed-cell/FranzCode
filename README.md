Windows:
1. Open the "Edit the system environment variables" menu in Windows.
2. Add the path to your franzcode folder to the Path variable.
3. You can now use the franz command in CMD or PowerShell

Interactive REPL
Type franz with no arguments to enter the interactive shell:
franz
>>> SET x TO 10
>>> SAY "{x} is the value"

Running a File
Create a file named hello.franz:
SET greeting TO "Hello World"
SAY greeting

Run it using:
franz hello.franz

Debugging Tools
1. View Tokens: franz --tokens file.franz (See how the lexer breaks down your code).
2. View AST: franz --ast file.franz (See the tree structure generated by the parser).
